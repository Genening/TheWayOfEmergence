### 模拟实验建议

1. **生命游戏** (Conway's Game of Life)
   - 体验简单规则产生复杂行为
   - 在线版本：playgameoflife.com

2. **鸟群模型** (Boids)
   - 理解集群行为的涌现
   - Processing/p5.js实现

3. **谢林隔离模型**
   - 轻微偏好如何导致严重隔离
   - NetLogo模型库有现成模型

4. **元胞自动机**
   - 探索不同规则的行为空间
   - Wolfram Alpha可在线实验

## 后续学习路径

### 如果你对理论感兴趣

```
第1步：阅读《系统之美》，建立系统思维
第2步：学习Complexity Explorer的入门课程
第3步：阅读《复杂》和《链接》
第4步：深入某个专题（网络、混沌、演化...）
```

### 如果你想动手实践

```
第1步：安装NetLogo，运行示例模型
第2步：修改现有模型，观察变化
第3步：设计自己的简单模型
第4步：用模型探索你感兴趣的问题
```

### 如果你想应用于工作

```
第1步：识别你工作中的复杂系统
第2步：用框架分析系统（元素、关系、涌现）
第3步：找出系统的杠杆点
第4步：设计小规模实验验证想法
第5步：逐步推广成功的干预
```
